# קסם האותיות ✨
### משחק לימוד אלפבית עברי לילדים

![Emily](assets/preview-badge.svg)

אפליקציית לימוד אינטראקטיבית לילדים בגן ובכיתה א׳, בליווי **אמילי** — הדמות המונפשת שמלווה את הילד לאורך כל המשחק.

---

## 🎮 מה יש במשחק?

| מסך | תיאור |
|-----|-------|
| 📖 **לומדים** | כל 22 אותיות האלפבית עם אמוג׳י ומילה — לחיצה משמיעה את האות |
| 🃏 **זיכרון** | משחק זיכרון — התאמת אות לאמוג׳י שלה |
| ✏️ **כתיבה** | ציור האות עם האצבע / עכבר על בד דיגיטלי |
| 🎯 **קווייז** | 8 שאלות — איזו אות מתחילה המילה? |
| 🎨 **צביעה** | ציור חופשי + דפי צביעה עם אות ואיור לצד |

---

## ✨ פיצ׳רים

- 🌸 **אמילי** — דמות מונפשת עם 4 פוזות (שמחה, חושבת, מופתעת, ניצחון)
- 🔊 **מוסיקה רגועה** — מבוססת על סקאלה פנטטונית + אפקטי קול
- 🎊 **קונפטי** — חגיגה בסיום משחק זיכרון ובקווייז מוצלח
- 💾 **שמירת ציורים** — הורדת הציור כ-PNG
- 📱 **רספונסיבי** — עובד על מובייל, טאבלט ודסקטופ
- 🇮🇱 **RTL** — ממשק עברי מלא

## 🚀 הפעלה

### מקומית
פשוט פתחו את `index.html` בדפדפן — אין צורך בשרת.

### GitHub Pages
1. העלו את הפרויקט ל-GitHub
2. הגדרות → Pages → Source: `main` branch, `/ (root)`
3. האתר יהיה זמין בכתובת `https://[username].github.io/[repo-name]/`
